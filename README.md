# Task Manager App

## 🚀 Overview

A full-stack task manager application where users can create, update, and delete tasks.

## 🛠 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express
* Database: (MongoDB / Local / etc.)

## ✨ Features

* Add tasks
* Edit tasks
* Delete tasks
* TasK due alert by sending mail
* Progess Graph
* Interactive UI


## 📦 Installation

### 1. Clone the repo

git clone https://github.com/TejasChaudhari-30/Task-Manager-App.git

### 2. Setup Backend

cd backend
npm install
npm start

### 3. Setup Frontend

cd frontend/my-react-app
npm install
npm run dev
